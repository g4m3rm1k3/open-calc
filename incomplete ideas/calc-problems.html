Chapter 1 — Epsilon–Delta

Reading ε–δ from a graph

When a problem says "use the graph to find δ for the given ε", here is the exact procedure — every time.

The method in plain English
You're given: a graph, a limit point a, the limit value L, and a value of ε.
You need to find: a δ such that the "ε–δ box" centered at (a, L) traps the function.

Procedure: Draw horizontal lines at L+ε and L−ε. Find where the graph exits those lines. The x-distances from a to those exit points give candidate δ values. Take the smaller one.
Worked Example 1.1
From the graph of f(x) = √(x+1), find δ such that |f(x) − 2| < 0.5 whenever |x − 3| < δ
1
Identify what you're given
a = 3  (the x-value the limit is approaching)
L = f(3) = √(3+1) = √4 = 2  (the limit value)
ε = 0.5  (the y-tolerance demanded)
2
Draw the horizontal ε-strips: y = L ± ε
Upper: y = L + ε = 2 + 0.5 = 2.5
Lower: y = L − ε = 2 − 0.5 = 1.5
On the graph, draw horizontal dashed lines at y = 2.5 and y = 1.5.
3
Find where the function hits those lines — solve for x
√(x+1) = 2.5 → x+1 = 6.25 → x = 5.25
√(x+1) = 1.5 → x+1 = 2.25 → x = 1.25
4
Measure the distance from a to each exit point
Right distance: |5.25 − 3| = 2.25
Left distance: |1.25 − 3| = 1.75
5
Take the SMALLER distance as δ
δ = min(2.25, 1.75) = 1.75
We take the smaller side because δ must work in both directions from a. If we used 2.25, the left side (at x = 3 − 2.25 = 0.75) would escape the ε band.
✓ Answer: δ = 1.75. For any x with |x − 3| < 1.75, we guarantee |f(x) − 2| < 0.5.

ε–δ Algebraically — the scratch-work method

For algebraic proofs you do two passes: scratch work (find what δ to use) and then a clean formal proof (verify it works). This is always the structure.

Worked Example 1.2
Prove using ε–δ that limx→3 (2x − 1) = 5
S
SCRATCH WORK — work backwards from what we need
We need: |f(x) − L| < ε, i.e. |(2x−1) − 5| < ε. Simplify the left side:
|(2x − 1) − 5| = |2x − 6| = 2|x − 3|
We need 2|x−3| < ε, so |x−3| < ε/2. So we choose δ = ε/2.
P
FORMAL PROOF — now write it cleanly forward
Given: Let ε > 0. Choose δ = ε/2.
Assume: 0 < |x − 3| < δ = ε/2
Then:
  |(2x − 1) − 5|
   = |2x − 6|
   = 2|x − 3|
   < 2 · (ε/2)
   = ε ✓
✓ Done. The chain ends with < ε. That's all the proof needs.
Worked Example 1.3 — harder
Prove limx→2 x² = 4
S
Scratch work — simplify |f(x) − L|
|x² − 4| = |(x−2)(x+2)| = |x−2| · |x+2|
We need this < ε. We control |x−2| (it's < δ). But what about |x+2|? We need to bound it. Assume δ ≤ 1 first — this is standard.
If |x − 2| < 1, then 1 < x < 3
So 3 < x + 2 < 5 → |x + 2| < 5
Now we have: |x² − 4| = |x−2|·|x+2| < δ · 5. We need δ · 5 < ε, so δ < ε/5. Choosing δ = min(1, ε/5) satisfies both constraints.
P
Formal proof
Given: Let ε > 0. Choose δ = min(1, ε/5).
Assume: 0 < |x − 2| < δ
Then: |x − 2| < 1 → |x + 2| < 5
  |x² − 4|
   = |x−2| · |x+2|
   < δ · 5
   ≤ (ε/5) · 5
   = ε ✓

The min(1, ε/5) trick appears in almost every quadratic or higher-degree ε–δ proof. You pick 1 (or any convenient number) to bound the variable factor, then set δ small enough to also give you ε/bound. Always take the min of the two.

Chapter 2 — Continuity

Proving a Function is Continuous

There's a three-part checklist. Miss any one and the function fails. Here's how to apply it systematically.

The 3-part test for continuity at x = a
1. f(a) is defined (no division by zero, no square root of negative, etc.)
2. limx→a f(x) exists (left limit = right limit)
3. limx→a f(x) = f(a) (the limit equals the actual value)

For piecewise: check each piece on its open interval (easy, just algebra), then carefully test the join points.
Piecewise Functions — the join point is everything

For a piecewise function, the interior of each piece is almost never the problem — polynomials and standard functions are continuous on their domains. The only place you need to check is where the pieces meet.

Worked Example 2.1
Is f continuous at x = 2?   f(x) = { x² − 1  if x < 2  |  3  if x = 2  |  2x − 1  if x > 2 }
1
Check f(a) exists
f(2) = 3  ✓  (explicitly defined)
2
Compute the LEFT limit (x → 2⁻, use the x < 2 piece)
limx→2⁻ f(x) = limx→2⁻ (x² − 1) = (2)² − 1 = 3
3
Compute the RIGHT limit (x → 2⁺, use the x > 2 piece)
limx→2⁺ f(x) = limx→2⁺ (2x − 1) = 2(2) − 1 = 3
4
Does the two-sided limit exist? Does it equal f(2)?
Left = Right = 3 → limx→2 f(x) = 3
f(2) = 3 = limit ✓
✓ f is continuous at x = 2. All three conditions hold.
Worked Example 2.2 — fails
Is g continuous at x = 1?   g(x) = { x + 1  if x ≤ 1  |  x² + 1  if x > 1 }
1
f(a): g(1) = 1 + 1 = 2 ✓
2
Left limit (x ≤ 1 piece)
limx→1⁻ (x + 1) = 2
3
Right limit (x > 1 piece)
limx→1⁺ (x² + 1) = 1 + 1 = 2
4
Check: Left = Right?
Left = 2, Right = 2  → equal!
So lim = 2 = f(1) = 2  ✓
✓ g IS continuous at x = 1 — both pieces approach 2.
Worked Example 2.3 — genuinely fails
Is h continuous at x = 0?   h(x) = { x²  if x < 0  |  x + 1  if x ≥ 0 }
1
h(0) = 0 + 1 = 1 ✓
2
Left limit
limx→0⁻ x² = 0
3
Right limit
limx→0⁺ (x + 1) = 1
4
Left ≠ Right → limit does not exist
0 ≠ 1  →  limx→0 h(x) does not exist
✗ h is NOT continuous at x = 0. It has a jump discontinuity.

Composition of Functions — the theorem you're thinking of

You're thinking of this theorem: if g is continuous at a, and f is continuous at g(a), then f ∘ g is continuous at a. It lets you build continuity from pieces.

The key insight you described: "left + left = right + right" — this is the squeeze on the join point. For a piecewise to be continuous, as you approach the join from the left, the left formula must give the same value as the right formula gives as you approach from the right. If they match, you can "join" them — the function is locally a single thing, and composition theorems apply on either side.

Worked Example 2.4 — composition
Find values of c making f continuous everywhere: f(x) = { cx² + 1  if x ≤ 2  |  x + c  if x > 2 }
1
Each piece is a polynomial — continuous on its domain automatically
We only need to worry about the join at x = 2.
2
Compute left and right limits in terms of c
limx→2⁻ f(x) = c(2²) + 1 = 4c + 1
limx→2⁺ f(x) = 2 + c = c + 2
3
Set left = right and solve for c
4c + 1 = c + 2
3c = 1
c = 1/3
4
Verify: also check f(2) = left limit
f(2) = c(4) + 1 = 4/3 + 1 = 7/3
lim = 4(1/3) + 1 = 7/3  ✓
✓ c = 1/3 makes f continuous everywhere.

This is the general method for "find c." Set left-limit = right-limit (which also = f(a) once c is chosen). You get one equation, solve for c. If you have two join points, you get a system of two equations.

Chapter 3 — Differentiability

Newton's Quotient and When it Fails

Differentiability is defined through Newton's quotient (the difference quotient). The derivative exists at a point if and only if this limit exists — and we can check it by comparing left and right limits.

Newton's Quotient / The Difference Quotient
f′(a) = limh→0 [f(a+h) − f(a)] / h

Equivalently: f′(a) = limx→a [f(x) − f(a)] / (x − a)

Both forms appear in problems. Know which to use based on what's cleaner.
Worked Example 3.1
Find f′(x) from the definition for f(x) = x²
1
Write the quotient
[f(x+h) − f(x)] / h = [(x+h)² − x²] / h
2
Expand the numerator
[(x² + 2xh + h²) − x²] / h
= [2xh + h²] / h
3
Cancel h (valid because h ≠ 0 in the limit)
= 2x + h
4
Take the limit as h → 0
limh→0 (2x + h) = 2x
✓ f′(x) = 2x for all x.
Worked Example 3.2
Is f(x) = |x| differentiable at x = 0?
1
Write the difference quotient at a = 0
[f(0+h) − f(0)] / h = [|h| − 0] / h = |h| / h
2
Evaluate left and right limits separately
As h → 0⁺: |h|/h = h/h = +1
As h → 0⁻: |h|/h = −h/h = −1
3
Left limit ≠ Right limit → no derivative
+1 ≠ −1 → limit does not exist
✗ f is continuous at 0 but NOT differentiable — the corner prevents a unique tangent slope.
Piecewise Differentiability — two conditions needed

Order of operations: To prove differentiability at a join point, you must FIRST verify continuity (differentiability implies continuity, so if it's not continuous it can't be differentiable). Then check that the derivatives from each side match.

Worked Example 3.3
Is f differentiable at x = 1?  f(x) = { x²  if x ≤ 1  |  2x − 1  if x > 1 }
1
Step 0: Check continuity first
f(1) = 1² = 1
limx→1⁻ x² = 1    limx→1⁺ (2x−1) = 1
All equal → continuous ✓
2
Left derivative: differentiate left piece, evaluate at join
(x²)′ = 2x  →  at x=1: f′⁻(1) = 2
3
Right derivative: differentiate right piece, evaluate at join
(2x−1)′ = 2  →  at x=1: f′⁺(1) = 2
4
Left derivative = Right derivative?
2 = 2  ✓
✓ f is differentiable at x = 1, with f′(1) = 2.
Worked Example 3.4 — fails differentiability
Is g differentiable at x = 0?  g(x) = { x²  if x ≤ 0  |  x  if x > 0 }
1
Continuity at x = 0
g(0) = 0,  limx→0⁻ x² = 0,  limx→0⁺ x = 0  ✓
2
Left derivative at x = 0
(x²)′ = 2x  →  at x=0: g′⁻(0) = 0
3
Right derivative at x = 0
(x)′ = 1  →  at x=0: g′⁺(0) = 1
✗ g is continuous but NOT differentiable at x = 0. Left slope = 0 (flat parabola), right slope = 1 (linear). Corner exists.
